> 총 길이: 2분 30초
> 대상: Moltiverse Hackathon 2026 심사위원

## Scene 1: 인트로 (0:00 - 0:15)

**화면:** Ghost Protocol 로고 + 태그라인 애니메이션
**내레이션:**
"Ghost Protocol에 오신 것을 환영합니다. AI 에이전트가 팩맨 아레나에서 대결하고, 관전자는 Monad 블록체인 위에서 실시간으로 베팅하는 — 완전히 새로운 AI 게이밍 플랫폼입니다."

**화면 전환:** 로고 → 메인 대시보드

---

## Scene 2: Arena Mode (0:15 - 1:00)

### 2-1. 토너먼트 브래킷 (0:15 - 0:25)
**화면:** 토너먼트 대시보드 — 8개 AI 에이전트 브래킷
**내레이션:**
"Arena Mode에서는 8개의 AI 에이전트가 싱글 엘리미네이션 토너먼트로 대결합니다. 각 에이전트는 고유한 전략을 가지고 있으며, 온체인에 등록된 신원을 갖습니다."

**행동:** 브래킷 확대 → 첫 번째 매치 선택

### 2-2. 실시간 매치 관전 (0:25 - 0:45)
**화면:** Phaser.js 게임 화면 — 2개 팩맨이 미로에서 경쟁
**내레이션:**
"매치가 시작되면 60fps 실시간으로 게임이 진행됩니다. 각 에이전트의 전략을 관전하면서 누가 더 효율적으로 펠릿을 수집하는지 지켜보세요."

**행동:** 게임 진행 하이라이트 (빠른 컷)

### 2-3. 베팅 플로우 (0:45 - 1:00)
**화면:** 사이드 패널 베팅 UI — 승패 배당률, 베팅 입력
**내레이션:**
"관전자는 Monad 위에서 실시간으로 베팅할 수 있습니다. 매치 시작 전 베팅 창이 열리고, 결과가 확정되면 스마트 컨트랙트가 자동으로 배당금을 정산합니다. 수수료는 단 5%입니다."

**행동:** 지갑 연결 → 베팅 금액 입력 → 트랜잭션 확인 → 승리 후 배당금 수령

---

## Scene 3: Survival Mode (1:00 - 1:45)

### 3-1. 게임플레이 (1:00 - 1:20)
**화면:** 서바이벌 게임 화면 — 플레이어 팩맨 vs AI 고스트 4마리
**내레이션:**
"Survival Mode에서는 사람이 직접 팩맨을 조작합니다. AI 고스트는 라운드가 올라갈수록 점점 더 똑똑해집니다."

**행동:** 라운드 1-2 (쉬운 난이도) → 라운드 5-6 (중간) → 라운드 8+ (매우 어려움)

### 3-2. 난이도 상승 시연 (1:20 - 1:30)
**화면:** 라운드 진행 오버레이 — 난이도 티어 표시
**내레이션:**
"Tier 1에서 고스트는 단순 순찰만 합니다. 하지만 Tier 4부터는 Claude AI가 전략을 수립하고, Tier 5에서는 매 틱마다 실시간으로 플레이어 행동을 분석합니다."

**행동:** 난이도 티어 비교 (사이드바이사이드 또는 빠른 컷)

### 3-3. 관전자 예측 베팅 (1:30 - 1:45)
**화면:** 예측 베팅 패널 — "몇 라운드까지 생존할까?"
**내레이션:**
"관전자는 플레이어가 몇 라운드까지 생존할지 예측 베팅합니다. 정확한 예측일수록 높은 가중 배당을 받습니다. 정확히 맞추면 3배, ±1 라운드면 2배 가중치가 적용됩니다."

**행동:** 예측 입력 → 라운드 진행 → 결과 확인 → 배당금 수령

---

## Scene 4: 온체인 검증 (1:45 - 2:05)

**화면:** Monad Block Explorer — 트랜잭션 기록
**내레이션:**
"모든 매치 결과와 베팅 정산은 Monad 블록체인에 기록됩니다. GhostArena 컨트랙트에서 토너먼트 결과를, WagerPool에서 베팅 정산을, SurvivalBet에서 예측 배당을 확인할 수 있습니다."

**행동:**
- GhostArena: 에이전트 등록 tx → 매치 결과 tx
- WagerPool: 베팅 tx → 정산 tx → 배당금 인출 tx
- SurvivalBet: 예측 베팅 tx → 결과 정산 tx

---

**화면:** Mermaid 다이어그램 또는 아키텍처 슬라이드
**내레이션:**
"Ghost Protocol은 결정론적 게임 엔진, 5단계 적응형 AI, WebSocket 실시간 스트리밍, 그리고 Monad 스마트 컨트랙트로 구성됩니다. Agent SDK를 통해 누구나 자신만의 AI 에이전트를 개발하고 토너먼트에 참여할 수 있습니다."

**행동:** 아키텍처 다이어그램 확대 → SDK 코드 예시 표시

## Scene 6: 클로징 (2:20 - 2:30)

**화면:** Ghost Protocol 로고 + 핵심 수치
**내레이션:**
"Ghost Protocol — AI와 블록체인이 만나는 새로운 게이밍의 미래. Monad 위에서 펼쳐지는 AI 에이전트 팩맨 아레나에 오신 것을 환영합니다."

**화면 표시:**
- 440+ 백엔드 테스트
- 227+ 스마트 컨트랙트 테스트
- 60fps 결정론적 엔진
- 5단계 AI 난이도
- 실시간 온체인 베팅

**엔드 카드:** GitHub 링크, 라이브 데모 링크

## 촬영 메모

### 필수 준비물
1. 로컬 개발 환경 실행 (프론트엔드 + 백엔드 + Redis)
2. MetaMask 또는 Rabby 지갑 (Monad 테스트넷 연결)
3. 테스트넷 MON 토큰 (faucet에서 수령)
4. 화면 녹화 소프트웨어 (OBS 또는 ScreenFlow)

### 녹화 팁
- 해상도: 1920x1080 (60fps 권장)
- 마우스 커서 강조 활성화
- 지갑 잔액이 보이도록 설정
- 트랜잭션 확인 시 Block Explorer 탭 미리 열어두기
- 게임 화면 녹화 시 음향 효과 활성화 (Tone.js 오디오)
